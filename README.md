# Machine Learning and Reinforcement Learning Projects

## 中文

### 项目内容

- 使用 KMeans 与 GMM（Gaussian Mixture Model）对数据进行聚类分析，并对两种方法进行比较研究。
- 采用 PCA（Principal Component Analysis）对聚类结果进行降维可视化展示。
- 使用 KNN（K-Nearest Neighbors）与 CNN（Convolutional Neural Network）进行预测任务，并对模型性能进行比较分析。
- 基于 SARSA 与 Q-Learning 算法完成迷宫导航任务，实现智能体自主路径探索与学习。

### 涉及技术

- KMeans Clustering
- Gaussian Mixture Model (GMM)
- PCA Visualization
- KNN Classification
- Convolutional Neural Network (CNN)
- SARSA Reinforcement Learning
- Q-Learning Reinforcement Learning

## English

### Project Description

- Performed data clustering using KMeans and Gaussian Mixture Model (GMM), followed by comparative analysis of both methods.
- Applied PCA (Principal Component Analysis) for dimensionality reduction and visualization of clustering results.
- Implemented KNN (K-Nearest Neighbors) and CNN (Convolutional Neural Network) for prediction tasks and compared their performance.
- Completed maze navigation tasks using SARSA and Q-Learning algorithms, enabling autonomous path exploration and learning for the agent.

### Technologies

- KMeans Clustering
- Gaussian Mixture Model (GMM)
- PCA Visualization
- KNN Classification
- Convolutional Neural Network (CNN)
- SARSA Reinforcement Learning
- Q-Learning Reinforcement Learning
